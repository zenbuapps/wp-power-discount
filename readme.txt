== Description ==

**Power Discount** 是一款專為台灣電商設計的 WooCommerce 折扣規則外掛，由 [Powerhouse](https://powerhouse.cloud) 開發。它把台灣常見的促銷模式（任選 N 件、紅配綠、第 N 件 X 折、滿額贈、條件免運）整合進一個高效能、可擴充的規則引擎，讓你不用再東拼西湊十幾個零散的外掛就能做出完整的促銷活動。

= 9 種折扣策略 =

* **商品折扣（Simple）** — 百分比折扣、固定金額折扣、指定售價。例：全站 9 折、指定商品折 $50。
* **數量階梯折扣（Bulk）** — 依購買數量套用不同折扣級別。例：1–4 件原價、5–9 件 9 折、10 件以上 8 折。
* **整車折扣（Cart）** — 整張購物車達門檻後，扣固定金額或百分比。例：滿千折百、整單 9 折。
* **任選 N 件組合（Set）** — 任選 N 件享組合價、組合折扣或現折金額。例：任選 2 件 $90、任選 4 件現折 $100。
* **買 X 送 Y（Buy X Get Y）** — 買 X 件就送 Y 件；贈品可以是同樣商品、最便宜的商品、或指定商品清單。
* **第 N 件 X 折（Nth item）** — 依購物車內商品的順序，第 N 件套用對應折扣。例：第二件 6 折、第三件半價。
* **紅配綠 / 跨類組合（Cross-category）** — 要求顧客同時購買多個分類的商品才能享折扣。例：上衣 + 褲子整組 8 折。
* **條件免運（Free shipping）** — 條件達成後免運、運費百分比折扣或運費固定金額折扣，支援指定運送方式。
* **滿額贈（Gift with purchase）** — 購物車達門檻自動加入贈品並折抵為 0 元。

= 13 種觸發條件 =

購物車小計、購物車總數量、購物車品項數、顧客累計消費、使用者角色、是否登入、付款方式、運送方式、日期區間、星期幾、時段、是否首次訂單、生日當月——全部可以用 AND / OR 組合使用。

= 6 種商品篩選 =

全部商品、指定商品、商品分類、商品標籤、商品屬性、特價商品，每個規則都可以組合多個篩選條件精準鎖定要套用折扣的商品。

= 可視化規則編輯器 =

* 不需要寫 JSON，所有設定透過表單完成
* 拖拉列表即可調整規則優先順序
* 每個規則都可以一鍵開關、複製、刪除
* 支援**每月重複排程**（例如每月 20 到 30 號自動啟用）
* 支援**套用後停止後續規則**（獨佔模式）

= 購物車與結帳頁整合 =

* 商品價格即時重新計算（相容經典購物車與 WooCommerce 區塊購物車）
* 結帳時產生折扣憑證並寫入訂單
* 購物車頁面顯示「已套用折扣」橘色通知框，讓顧客清楚看到省了多少
* 滿額贈支援進度條提示

= 折扣統計報表 =

內建 Reports 頁面，可依日期區間查詢每條規則的套用次數與總折扣金額，方便行銷團隊追蹤促銷成效。

= 台灣電商導向 =

* 預設繁體中文介面
* 折扣金額以 NT$ 顯示
* 台灣零售常見的折扣術語（9 折、滿千折百、紅配綠）直接對應
* 與綠界、藍新金流等台灣主流金流無衝突
* 支援 WooCommerce HPOS（High-Performance Order Storage）

== Installation ==

1. 下載 `power-discount.zip` 後到 WordPress 後台 → 外掛 → 安裝外掛 → 上傳外掛。
2. 啟用外掛。
3. 左側選單點選「PowerDiscount」→「折扣規則」，開始新增你的第一條規則。

== Frequently Asked Questions ==

= 這個外掛跟 WooCommerce 內建的優惠券衝突嗎？ =

不會。Power Discount 是透過 WooCommerce 計價流程直接調整商品單價與建立折扣 fee，和優惠券是兩套獨立機制，可以同時使用。

= 可以在購物車與結帳頁同時顯示折扣嗎？ =

可以。外掛會在購物車與結帳頁下方自動顯示「已套用折扣」通知框，列出每條規則套用的金額與套用到的商品。

= 支援 WooCommerce 區塊購物車嗎？ =

支援。外掛同時相容經典購物車與區塊購物車（Store API），訂單建立後折扣會寫入 order line item meta。

= 可以匯出折扣套用紀錄嗎？ =

目前 Reports 頁面可依日期區間查詢統計。未來版本會加上 CSV 匯出功能。
